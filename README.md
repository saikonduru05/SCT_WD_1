This project is a Responsive Landing Page built using HTML, CSS, and JavaScript.
It demonstrates modern web design principles with a clean and interactive navigation bar.
The navigation menu changes style when the user scrolls the page, improving user experience.
The layout is fully responsive and adapts to different screen sizes such as mobile, tablet, and desktop.
Smooth scrolling and section-based navigation make the page easy to explore.
This project helps in understanding DOM manipulation, responsive design, and UI layout structuring.
FEATURES:
Responsive navigation bar
Scroll-based styling effects
Clean and modern UI
Mobile-friendly layout
Simple and structured code
Technologies Used: HTML, CSS, JavaScript
